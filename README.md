# Bike Signal Lights

v0.1
Small project made using Raspberry Pi Pico and MicroPython.

## Basic Components
* 1x Raspberry Pi Pico
* 1x 4-8x8 LED Matrix with MAX7219
* 1x KY-023 Joystick Module (for prototype version)
* 1x Grove - LED Button
* Jumper Cables
